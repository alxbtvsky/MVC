MVC

by Alex Batovsky

Posted on March 24, 2015 at 9:00 PM

Introduction:

The objective of this project was the use MVC.NET and bootstrap to implement the wireframe below. I had the freedom to be creative in every aspect, as long as all fields were shown and able to be updated after hitting the 'change' button. Authentication was not required, but validation had to be done on all fields. All code had to be checked into Github and deployed into azure, but the code did not have to persist to a database.

Digging Deeper:

Using MVC.NET and bootstrap to do this project made it substantially less difficult. MVC.NET and bootstrap do a lot for you right out of the gate. For instance, when you first create your project in MVC.NET, bootstrap is already in place and a template for your entire website is already in place, as well. It is then your job to create the look and feel of your website, while adding additional features and functionality.
